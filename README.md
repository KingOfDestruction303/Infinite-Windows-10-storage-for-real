# Infinite-Windows-10-storage-for-real
get 64 tb storage (and near infinite storage) using windows 10 cluster size with vhd and storage spaces this works 100% percent even in windows 11

steps :
open this pc 
go to manage
go to disk management > action > create vhd > dynamic and vhdx and choose whatever size you like up to 64 tb 
go to the created disk > right click > initilize disk > ok > next 3x change defauly to 1mb > next > finish
go to storage spaces > create storage space > single on resilliencey > create storage space
Done you now have near infinite storage
